# FASTA Protein Analysis Toolkit

A command-line tool to analyse protein sequences from a FASTA file and export per-protein statistics to CSV.

## Features

- Parses any FASTA file using Biopython
- Computes per-protein statistics: length, molecular weight, isoelectric point
- Handles ambiguous residues (e.g. `X`) gracefully without crashing
- Exports results to a CSV file
- Prints summary statistics for the dataset
